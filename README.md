## 🚀 快速开始

### 📋 环境要求
- 现代浏览器（Chrome 88+、Firefox 85+、Safari 14+、Edge 88+）
- 支持WebSocket和Web Speech API
- 网络连接（用于AI翻译服务）

### 🛠️ 安装部署

```bash
# 克隆项目
git clone https://github.com/your-username/smt-translator.git

# 进入项目目录
cd smt-translator

# 直接在浏览器中打开 index.html
# 或使用本地服务器
python -m http.server 8000
# 然后访问 http://localhost:8000
```

### ⚡ 一键启动

只需在浏览器中打开 `index.html` 文件，即可立即开始使用！

## 📖 使用指南

### 🔤 文本翻译
1. **输入文本** - 在左侧文本框输入需要翻译的内容
2. **选择语言** - 点击语言按钮选择源语言和目标语言
3. **开始翻译** - 点击"翻译"按钮或使用快捷键 `Ctrl/Cmd + Enter`
4. **复制结果** - 点击"复制"按钮将翻译结果复制到剪贴板

### 🎵 语音朗读
1. **基础朗读** - 翻译完成后点击"朗读"按钮
2. **语音设置** - 点击"朗读"按钮旁的⚙️图标打开设置面板
3. **个性化调节** - 调整语速、音调、音量，选择喜欢的语音
4. **测试语音** - 在设置面板中点击"测试语音"预览效果

### 📷 图片翻译
1. **上传图片** - 点击"图片翻译"或直接拖拽图片到指定区域
2. **OCR识别** - 系统自动识别图片中的文字
3. **查看结果** - 识别的文字将自动填入翻译框
4. **一键翻译** - 点击翻译按钮获得翻译结果

### ⌨️ 快捷键
- `Ctrl/Cmd + Enter` - 快速翻译
- `Ctrl/Cmd + L` - 切换语言
- `Escape` - 关闭设置面板

- **前端框架**: 原生JavaScript + HTML5 + CSS3
- **AI引擎**: 讯飞星火认知大模型
